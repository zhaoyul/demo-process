## 五、服务接口

| 接口 | 协议 | 用途 |
|------|------|------|
| 仿真提交 | REST POST | 提交 .i 文件, 返回 job_id |
| 结果查询 | REST GET | 按 job_id 获取 .e / .csv |
| 衍生量推送 | MQ (RabbitMQ) | 推送后处理器指标 |
| 实时监控 | WebSocket | 求解进度、残差推送 |
| 健康检查 | REST GET /health | 节点状态 |
